# Food-Search-Engine

Welcome to food search engine developed using ember.js and Edamam public food API.
Using this engine, user can search all the food dishes made using entered item in input box.
Clicking on 'Get Food Dishes' would render all the food items.
User can search food items by entering food content in table input box.
Emptying input box will revert food list items to it's orginal state.
Lot of functionalities can be added but due to time constraint i just implemented basic functions of the app.

Follow below instructions to run this project in your local machine.

## Prerequisites

You will need the following things properly installed on your computer.

* [Git](https://git-scm.com/)
* [Node.js](https://nodejs.org/) (with npm)
* [Ember CLI](https://ember-cli.com/)
* [Google Chrome](https://google.com/chrome/)

## Installation

* `git clone <repository-url>` this repository
* `cd search-engine`
* `npm install`

## Running / Development

* `ember serve`
* Visit your app at [http://localhost:4200](http://localhost:4200).
* Visit your tests at [http://localhost:4200/tests](http://localhost:4200/tests).
